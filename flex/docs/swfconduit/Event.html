Classpublic class Event
InheritanceEvent flash.events.Event

The Event is a message between server and client. Every message between server and client is handled by an Event class. The Event object encapsulates the objects being transferred. You should subclass this for your own events. Make sure to use flash.net.registerClassAlias() to register your classes.
