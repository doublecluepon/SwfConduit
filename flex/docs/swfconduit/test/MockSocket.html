Classpublic class MockSocket
InheritanceMockSocket flash.events.EventDispatcher
Implements ISocket

This is a mock socket that implements SwfConduit's socket interface, allowing for testing without a server. Instead of sending events over a socket, this object captures them and adds them to an internal array which can be examined and then cleared for the next event. To test events being sent from the server, use recieveEvent(), which will send the given event through the normal dispatch cycle.
